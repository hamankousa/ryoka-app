# プライバシーポリシー（ドラフト）

最終更新日: 2026-02-17

この文書は暫定版です。正式版は後日更新します。

## 1. 取得する情報

- 本アプリは、氏名・メールアドレス等の個人情報をアプリ内で直接収集しません。
- 設定値（テーマ、表示設定、フィルタ挙動など）とダウンロード済みデータは端末内に保存されます。

## 2. 利用目的

- アプリ機能の提供（再生、表示、オフライン利用）
- ユーザー設定の保持
- 品質改善のための運用（将来導入時は本ポリシーを更新）

## 3. 外部サービス

- コンテンツ配信・ホスティング等で外部サービスを利用する場合があります。
- それらサービス側で通信ログ等が処理される可能性があります。

## 4. 端末内データの削除

- ダウンロード済みデータは、アプリ内の削除操作で端末から削除できます。
- アプリをアンインストールすると、端末内のアプリデータは削除されます（OS仕様による）。

## 5. 変更

- 本ポリシーは必要に応じて更新します。
- 更新後の内容は本ファイルに反映します。
